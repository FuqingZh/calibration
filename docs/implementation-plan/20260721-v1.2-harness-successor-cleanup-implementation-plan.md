## Slice 2: Staged Comparative Evaluation

Status: closed — Stage A passed; Stage B failed

The evaluation froze the accepted baseline and successor source with the same
model, medium reasoning effort, fixtures, turn protocol, three repetitions,
isolated per-session homes, system-skill snapshot, seeded schedule, and blind
mapping. The complete 19-case run was fingerprinted before either stage ran.

### Stage A: targeted cases

The six targeted cases ran for both arms: 36 sessions and 18 blind pairs.
Stage A passed all continuation gates, including 18/18 candidate critical
passes and 3/3 passes for `harness-human-escalation`.

### Stage B: regression cases

The remaining 13 cases ran only after Stage A passed: 78 additional sessions
and 39 blind pairs. Stage B rejected the successor because candidate required
coverage was 94.0% versus 95.7% for the baseline, and candidate pairwise wins
were 15 versus 20 baseline wins. Critical gates did not regress: both arms had
three failures and there was no baseline-pass/candidate-fail critical pair.

Time, turns, and token usage remained secondary measures and did not override
the quality-gate failures.

### Blind review

A fresh independent task received only blind pairs, rubrics, the blind
manifest, and an empty scorecard. It completed all 18 Stage A rows before the
continuation decision and all 39 Stage B rows before arm identity was revealed.
